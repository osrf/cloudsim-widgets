# Usage:

1. Install dependency

    sudo pip install requests_oauthlib

2. Run script

    python example.py

3. Copy given URL into browser

4. Grant access

5. OH YES

6. Copy "OH YES"'s URL back into the terminal

7. Information about all your repos will be printed, including private ones
